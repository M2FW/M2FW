> Helper to create basic CRUD functions for resources which is running on M2FW ecosystem

## Install

```sh
yarn install
```

## Build

```sh
yarn build
```

## Build for development

```sh
# Watching source chages to rebuild it
# You can `yarn link` to the place where you want to use it
yarn build:dev
```
